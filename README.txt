File Organization
> Data\
>>> \DB_in\              : Database inclusive of roll dates, full data base, does not include benchmark
>>> \DB_ex\              : Database exclusive of roll dates, limited data base, includes benchmark
>>> \Aux\                : Auxillary data including external variables
>>> \Previous\           : Previous iterations of the database
> Analysis\
>>> \Graphs\             : Base graphs, figures for prices, returns, TDs
>>> \Summary_statistics\ : Summary statistics calculations with csv outputs
>>> \CAPM\               : Mincer-Zarnowitz regression with results inclusing CSV of all residuals
>>> 
> Papers\                : Papers, books, and resources
